Haiilo Enhancer

Extension Status

Temporarily turn off all extension features (muting, avatar styling, messenger expansion, date overrides) without uninstalling the extension.

Muting Options

Right-click any username on Haiilo and choose Haiilo Enhancer → Mute for …. Muted users' posts and comments are hidden on every page until they expire. Unmute them from this extension's toolbar popup, or via the context menu, or by adding them again with "Permanently" off and waiting it out.

Date & Time Format

Override Haiilo's default date and time display. Useful if your account language is set to English but you'd prefer European or 24-hour formatting. Affects only this browser.

Choose how dates are displayed (e.g., 01/15/2026 vs 15/01/2026)
Choose between 12-hour (AM/PM) and 24-hour time display.

Custom Homepage Settings

Set which homepage the logo redirects to for each Haiilo instance. Right-click on a homepage tab (e.g., "Home", "Home (soft)") and select "Set as default homepage" to configure.

Auto-Expand Sidebar Lists

Haiilo's left sidebar shows a limited number of Workspaces and Pages before hiding the rest behind a "Show more" button. Enable this to auto-click that button on page load so all entries are visible at once.

Choose which sidebar lists the auto-expander should affect.
How many times to click "Show more" for each list (Workspaces and Pages are expanded independently). 0 disables clicking for that list. Max 10.
How long to wait between consecutive "Show more" clicks. Lower values feel snappier, higher values are gentler on the server. Range: 100-1000 ms.

Custom Haiilo Domains

If your organization uses a custom Haiilo URL (not *.haiilo.app or *.haiilo.com), add it here. The browser will ask you to grant access the first time — that's expected, it just means "yes, run on this site too."

Enter just the domain name (e.g., intranet.company.com) Note: Due to a Chrome limitation, domain permissions cannot be revoked after being granted. Removing a domain here stops the extension from using it, but the permission will remain visible in chrome://extensions until you reinstall the extension.

Data Management

Export your settings and muted users to a JSON file as a backup, or import a previous export to restore them. Reset to Defaults wipes everything; export first if you might want to undo it.
